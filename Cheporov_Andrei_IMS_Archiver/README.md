
# Архиватор IMS — CHEPOROV ANDREI

## Описание

Консольное кроссплатформенное приложение на C++ для архивации и распаковки одного файла.

- Поддерживает `SIGINT (Ctrl+C)` — прерывание операции
- Работает с большими файлами (10 ГБ+)
- Использует `zlib` (через `gzopen` и `gzread/write`)
- Поддержка Windows и Linux
- Сборка через CMake

## Сборка

```bash
mkdir build
cd build
cmake ..
make
```

## Использование

```bash
./myapp a input.txt output.gz     # архивировать
./myapp e output.gz output.txt    # распаковать
```


Можно прервать в любой момент с помощью `Ctrl+C`. Приложение корректно остановится.
